				Uso de la GUI


Vim funciona en una terminal normal.  GVIM puede hacer las mismas cosas y
algunas más.  La GUI ofrece menús, una barra de menú, barras de
desplazamiento y otros.  Este capítulo es sobre las funcionalidades extra.

*09.1*	Partes de la GUI

Puede que tengas un icono en tu escritorio que arranca gVim.  En caso
contrario, uno de estos comandos lo hará: >

	gvim archivo.txt
	vim -g archivo.txt

Si esto no funciona es que no tienes la versión de Vim con soporte para GUI.
Tendrás que instalar una primero.
   Vim abrirá una ventana y mostrará "archivo.txt".  Cómo sea la ventana
dependerá de la versión de Vim.  Se parecerá a la siguiente figura (¡tanto como
esto se puede parecer con texto ASCII!).

	+----------------------------------------------------+
	| archivo.txt + (~/dir) - VIM			   X |	<- título
	+----------------------------------------------------+
	| File	Edit  Tools  Syntax  Buffers  Window  Help   |	<- barra de
	+----------------------------------------------------+     menú
	| aaa  bbb  ccc  ddd  eee  fff	ggg  hhh  iii  jjj   |	<- barra de
	| aaa  bbb  ccc  ddd  eee  fff	ggg  hhh  iii  jjj   |     herramientas
	+----------------------------------------------------+
	| file text					 | ^ |
	| ~						 | # |
	| ~						 | # |	<- barra de
	| ~						 | # |     desplazam.
	| ~						 | # |
	| ~						 | # |
	|						 | V |
	+----------------------------------------------------+

El espacio más grande está ocupado por el archivo de texto.  Esto muestra el
archivo de una manera similar a una terminal.  Con algunas diferencias de
fuente y color, quizá.


EL TÍTULO DE LA VENTANA

En la parte superior se encuentra el título.  Esto es dibujado por tu sistema
de ventanas.  Vim pondrá en título el nombre del archivo actual.  Primero
viene el nombre del archivo, luego algunos caracteres especiales y el
directorio del archivo entre paréntesis.  Los caracteres especiales son:

	-	El archivo no puede ser modificado (por ej. la ayuda)
	+	El archivo contiene cambios
	=	El archivo es de sólo lectura
	=+	El archivo es de sólo lectura pero contiene cambios

Si no hay ningún símbolo se trata de un archivo normal, sin cambios.


LA BARRA DE MENÚ

Vim tiene los elementos habituales, con algunos más.  Explóralos para tener
una idea de cómo podrías usarlos.  Un submenú relevante es Edit/Global
Settings.  Encontrarás los siguientes elementos:

	Toggle Toolbar		barra de herramientas aparece/desaparece
	Toggle Bottom Scrollbar	barra de despl. aparece/desaparece abajo
	Toggle Left Scrollbar	barra de despl. aparece/desaparece a la izq.
	Toggle Right Scrollbar	barra de despl. aparece/desaparece a la der.

En la mayoría de los sistemas puedes arrancar los menús.  Selecciona el
elemento superior del menú que parece una línea a rayas.  Ahora tendrás una
ventanita aparte con el menú.


LA BARRA DE HERRAMIENTAS

Esto contiene los iconos para las acciones más empleadas.  La intención es que
los iconos se expliquen a sí mismos.  Hay globos de ayuda que te aconsejan
(pon el puntero sobre un botón y no lo muevas durante un segundo).

La opción "Edit/Global Settings/Toogle Toolbar" puede ser empleada para hacer
que la barra de herramientas desaparezca.  Si no vas a querer una barra de
herramientas nunca, pon esto en tu vimrc: >

	:set guioptions-=T

Esto elimina la bandera 'T' de la opción 'guioptions'.  Otras partes del GUI
también pueden ser activadas o desactivadas del mismo modo. Consulta la ayuda.


LAS BARRAS DE DESPLAZAMIENTO

Por defecto sólo hay una barra de desplazamiento en la derecha.  Hace lo que
se espera de ella.  Cada ventana tendrá su barra de desplazamiento.
   Puedes hacer que una barra horizontal aparezca con la opción Edit/Global
Settings/Toggle Bottom Scrollbar.  Esto es útil en el modo diff, o cuando la
opción 'wrap' está desactivada (eso lo veremos después).

Cuando hay ventanas divididas verticalmente, sólo las de la derecha tendrán
barra de desplazamiento.  Sin embargo, si mueves el cursor a una ventana en la
izquierda, será esta la que la barra controle.  Lleva cierto tiempo
acostumbrarse a esto.
   Cuando trabajes con ventanas divididas verticalmente, considera la
posibilidad de añadir barras en la izquierda.  Prueba con esta opción: >

	:set guioptions+=1

Esto añade la bandera 'l' a 'guioptions'.

*09.2*	Uso del ratón

Los estándares son maravillosos.  En Microsoft Windows, puedes usar el ratón
para seleccionar texto de una forma estándar.  El sistema X de ventanas
también tiene un estándar para el uso del ratón.  Por desgracia, estos dos
estándares no son compatibles.
   Afortunadamente, puedes personalizarlo en Vim.  Puedes hacer que el
comportamiento del ratón sea el mismo que el de las X o el de Windows.  El
siguiente comando hace que el ratón se comporte como en las X: >

	:behave xterm

El siguiente comando hace que el ratón se comporte como en Windows: >

	:behave mswin

El comportamiento por defecto de un ratón en los sistemas UNIX es xterm.  El
comportamiento por defecto en un sistema Windows es seleccionado durante la
instalación.  Para más detalles sobre estos comportamientos, ver |:behave|.
Esto es un pequeño resumen:


COMPORTAMIENTO DEL RATÓN XTERM

Click botón izquierdo		posicionar el cursor
Arrastrar botón izquierdo	seleccionar texto en modo Visual
Click botón central		pegar texto desde el portapapeles
Click botón derecho		extender el texto seleccionado hasta el
				puntero del ratón


COMPORTAMIENTO DEL RATÓN MSWIN

Click botón izquierdo		posicionar el cursor
Arrastrar botón izquierdo	seleccionar texto en modo Seleccionar (|09.4|)
Click botón izquierdo, con May. extender el texto seleccionado hasta el
				puntero del ratón
Click botón central		pegar texto desde el portapapeles
Click botón derecho		mostrar un menú pop-up


El ratón puede ser ajustado con más detalle.  Échale un vistazo a estas
opciones para ver cómo funciona el ratón:

	'mouse'			en qué modo el ratón se usa por Vim
	'mousemodel'		qué efectos tiene hacer click con el ratón
	'mousetime'		tiempo entre dos clicks para un doble click
	'mousehide'		esconde el ratón mientras tecleas
	'selectmode'		el ratón comienza el modo Visual o Seleccionar

*09.3*	El portapapeles

En la sección |04.7| se explica el uso básico del portapapeles.  Hay algo
esencial que debe ser explicado sobre las X-windows: Hay dos lugares para
intercambiar texto entre programas.  MS-Windows sólo tiene uno.

En las X-windows hay una "selección actual".  Esto es el texto que está
actualmente marcado.  En Vim esto es el área Visual (asumiendo que uses las
opciones por defecto).  Puedes pegar esta selección en otra aplicación sin
ninguna acción ulterior.
   Por ejemplo, en este texto selecciona unas cuantas palabras con el ratón.
Vim cambiará al modo Visual y marcará el texto.  Ahora comienza otro gVim, sin
un nombre de archivo, de forma que muestre una ventana vacía.  Haz click con
el botón central del ratón.  El texto seleccionado será insertado.

La "selección actual" sólo permanecerá válida hasta que otro texto sea
seleccionado.  Tras pegar en otro gVim, ahora selecciona algunos caracteres en
esa ventana.  Te darás cuenta de que las palabras que estaban previamente
seleccionadas en otra ventana gVim se muestran de otro modo.  Esto significa
que ya no están en la "selección actual".

No necesitas seleccionar texto con el ratón, el uso de comandos del teclado
para el modo Visual funciona igual.


EL VERDADERO PORTAPAPELES

Llamamos al otro portapapeles el "verdadero portapapeles", para evitar la
confusión.  A menudo la "selección actual" y el "verdadero portapapeles" son
denominados indistintamente "portapapeles".  Tendrás que acostumbrarte a esto.
   Para poner el texto en el verdadero portapapeles, selecciona unas palabras
en uno de los gVims que tengas funcionando.  Usa la entrada de menú Edit/Copy.
El texto será copiado al verdadero portapapeles.  No podrás comprobar esto a
menos que tengas alguna aplicación que muestre el contenido del portapapeles.
   Ahora selecciona otro gVim, coloca el cursor en algún lugar y usa el menú
Edit/Paste.  Verás que se inserta el texto del verdadero portapapeles.


USAR AMBOS

Esto de usar simultáneamente la "selección actual" y el "verdadero
portapapeles" puede sonar un poco confuso.  Pero es muy útil.  Veámoslo con un
ejemplo.  Abre un gVim con un archivo de texto y haz lo siguiente:

-  Selecciona dos palabras en modo Visual.
-  Pon esas palabras en el portapapeles con Edit/Copy.
-  Selecciona otra palabra en modo Visual.
-  Accede al elemento del menú Edit/Paste.  Lo que ocurrirá es que la palabra
   seleccionada será sustituida con las palabras del portapapeles.
-  Pon el puntero del ratón en algún lugar y haz click con el botón central.
   Verás que la palabra que sobreescribiste con el portapapeles se insertará
   en ese lugar.

Si usas la "selección actual" y el "verdadero portapapeles" con cuidado,
puedes llevar a cabo una gran variedad de trucos de edición.


USO DEL TECLADO

Si no te gusta usar el ratón, puedes acceder a la selección actual y al
verdadero portapapeles con dos registros.  El registro "* es para la selección
actual.
   Para hacer que el texto se convierta en selección actual, usa el modo
Visual.  Por ejemplo, para seleccionar una línea completa usa "V".
   Para insertar la selección actual antes del cursor: >

	"*P

Comprueba que es una "P" mayúscula.  La "p" minúscula la pone tras el cursor.

El registro "+ se usa para el verdadero portapapeles.  Por ejemplo, para
copiar el texto desde la posición del cursor hasta el final de línea: >

	"+y$

Recuerda, "y" es copiar (yank).
   Para insertar los contenidos del verdadero portapapeles antes del cursor: >

	"+P

Es lo mismo que para la selección actual, pero usa el registro (+) en lugar
del registro estrella (*).

*09.4*	Modo Seleccionar

Esto algo que es empleado más habitualmente en MS-Windows que en X-Windows.
Aún así, ambos pueden hacerlo.  Ya conoces el modo Visual.  El modo
Seleccionar es como el modo Visual, también empleado para la selección.  Pero
hay una diferencia básica: Cuando escribas texto, el texto seleccionado se
elimina y el texto introducido lo sustituye.

Para iniciar el modo Seleccionar, debes activarlo primero (para MS-Windows
está probablemente activado, pero puedes hacer esto igualmente): >

	:set selectmode+=mouse

A continuación usa el ratón para seleccionar algún texto.  Será coloreado como
con el modo Visual.  Ahora pulsa un carácter.  El texto seleccionado se
eliminará, y ese carácter lo reemplazará.  Entrarás en modo Insertar.

Ya que escribir texto normal hace que el texto seleccionado se elimine, no
podrás usar los comandos de movimiento "hjkl", "w", etc.  En su lugar, usa las
teclas de las flechas con mayúsculas.  <S-izquierda> mueve el cursor a la
izquierda.  El texto seleccionado cambiará como en el modo Visual.  <S-Fin> y
<S-Inicio> también funcionan.

Puedes ajustar cómo el modo Seleccionar funciona mediante 'selectmode'.
